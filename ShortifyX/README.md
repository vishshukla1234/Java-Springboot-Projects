# 🚀 ShortifyX

A modern, secure, and scalable URL Shortener platform built using **Spring Boot**, **React**, and **PostgreSQL**. ShortifyX allows users to generate shortened URLs, create custom aliases, track link analytics, and manage their links through an intuitive dashboard.

---

## ✨ Features

- 🔗 Shorten long URLs instantly
- 📊 View click analytics
- 🔒 Secure JWT Authentication & Authorization
- 👤 User registration and login
- 📱 Responsive UI for desktop
- 🐳 Dockerized backend for easy deployment

---
